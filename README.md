Tic Tac Toe Game

This project is a simple and interactive Tic Tac Toe game developed using HTML, CSS, and JavaScript. It demonstrates fundamental web development concepts along with basic game logic such as turn-based interaction, winner detection, and draw conditions.

Features:-
-3x3 interactive game board
-Two-player mode (Player X and Player O)-
-Automatic turn switching
-Winner detection logic
-Draw condition detection
*Restart button to reset the game
*Runs directly in the browser without any installation
*Technologies Used
-HTML for structure
-CSS for styling and layout (Grid)
-JavaScript for game logic and interactivity
*How to Run
Download or clone this repository
Open the index.html file in any web browser
Start playing the game
*Working of the Game
The game starts with Player X
Players take turns selecting empty cells
After each move, the system checks for a winning condition
If a player aligns three marks in a row, column, or diagonal, that player wins
If all cells are filled and no winning condition is met, the game results in a draw
Game Output
Start of Game

The game begins with an empty 3x3 grid and displays "Player X's Turn", indicating that Player X will make the first move.

Player X Wins

The system detects a winning pattern when Player X aligns three marks in a row, column, or diagonal. The message updates to "Player X Wins!" and further moves are disabled.

-Draw Condition:-
When all cells are filled and no player achieves a winning pattern, the game declares a draw by displaying "Draw!". No further moves are allowed.
